Dokumentasi ini disusun untuk tim engineering Ciheul Technologies. Bagi yang baru masuk, Bootcamp dapat digunakan untuk mengakselerasi kemampuan teknis agar dapat memenuhi standar Ciheul Technologies. Bagi yang ingin mendokumentasikan knowledge yang didapat terkait snippet code dari stackoverflow hingga troubleshooting konfigurasi sangat dianjurkan untuk berbagi di Bootcamp. 

Bootcamp ditulis menggunakan Sphinx Documentation dengan format reStructured Text (reST) dan dapat diunduh di `http://github.com/ciheul/bootcamp <http://github.com/ciheul/bootcamp>`_.

Perhatian bagi yang ingin berkontribusi di Bootcamp untuk tidak mengunggah informasi sensitif seperti password.
